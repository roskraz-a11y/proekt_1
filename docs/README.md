# Документация проекта

## Структура проекта

```
proekt_1/
├── backend/          # PHP бэкенд
│   ├── index.php     # Главная точка входа API
│   └── config.php    # Конфигурация
├── frontend/         # Фронтенд
│   ├── index.html    # HTML страница
│   ├── css/
│   │   └── style.css # Стили
│   └── js/
│       ├── api.js    # API клиент
│       └── main.js   # Основной скрипт
├── README.md         # Описание проекта
└── .gitignore        # Git конфиг
```

## Как запустить

### Фронтенд
1. Откройте `frontend/index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   cd frontend
   python -m http.server 3000
   ```

### Бэкенд
1. Убедитесь, что у вас установлен PHP
2. Запустите встроенный сервер PHP:
   ```bash
   cd backend
   php -S localhost:8000
   ```
